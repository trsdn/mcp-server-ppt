# 4. Framework (2×2, Venn, Value Chain)

**When:** Organizing concepts, showing relationships, strategic positioning.

**Layout (2×2 matrix):**
```
Title: top
Matrix: centered (x=180pt, y=110pt, w=600pt, h=360pt)
  - Quadrant labels inside each quadrant
  - Axis labels outside
  - Items plotted as circles or icons
```

**Layout (Venn):**
```
Two circles: overlap at center
Labels inside each section
Intersection label in overlap zone
```

**Layout (Strategic Pillars — 3 columns):**
The most common consulting framework layout. Used for strategies, capabilities, initiatives.
```
Vision bar: full width (x=36, y=85, w=888, h=50), Primary fill, white text, centered
Pillar cards: 3 equal columns below (use 3-column grid from layout-grids.md)
  Each card:
    - Number badge: colored circle (50x50pt) centered at top
      Pillar 1: Primary color, Pillar 2: Accent color, Pillar 3: Positive color
    - Title: centered, 16pt bold, same color as badge
    - Bullet points: 2-3 items MAX per pillar (not 4+), 12pt, left-aligned with • markers
    - Card background: Neutral color (#F0F4F8)
```

CRITICAL: Pillar cards must have 2-3 bullets each, NOT 4+. Three pillars x 4 bullets = 12 bullets = wall of text. Keep each pillar to its 2 strongest initiatives.
