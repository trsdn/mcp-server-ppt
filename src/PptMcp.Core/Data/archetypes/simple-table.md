# 5. Simple Table

**When:** Comparing structured data, financials, feature matrices.

**Layout:**
```
Title: top
Table: centered (x=80pt, y=110pt, w=800pt, h=auto)
  - Header row: bold, brand color background, white text
  - Alternating row shading (subtle, 5% grey)
  - Right-align numbers, left-align text
Source: bottom
```

**Rules:**
- Max 6 columns, 8 rows
- Use horizontal lines only (no vertical grid lines)
- Highlight key row/column with accent color
