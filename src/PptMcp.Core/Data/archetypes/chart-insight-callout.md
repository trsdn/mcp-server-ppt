# 3. Chart + Insight Callout

**When:** Data visualization needs a "so what" annotation.

**Layout:**
```
Title: top (action title states the insight)
Chart: left 55% (x=50pt, y=100pt, w=500pt, h=340pt)
Callout box: right 40% (x=580pt, y=160pt, w=330pt, h=220pt)
  - 2-3 bullet insights
  - Key number highlighted
Source: bottom
```
