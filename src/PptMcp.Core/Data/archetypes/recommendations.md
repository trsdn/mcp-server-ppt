# 11. Recommendations / Next Steps

**When:** Closing a presentation with actions.

**Layout (structured — preferred):**
```
Title: top (action title: "Three initiatives to launch in Q3")
Table-style rows: 3 items, each with structured columns:

  Row format:
  ┌─────────┬──────────────────────────────────┬─────────────────┬──────────────┐
  │ # Badge │ Action (bold, 14pt)              │ Owner           │ Timeline     │
  │  (40pt) │ Impact description (12pt, grey)  │ (12pt, grey)    │ (12pt, grey) │
  └─────────┴──────────────────────────────────┴─────────────────┴──────────────┘
  Divider line between rows

Column widths: badge 50pt, action 500pt, owner 180pt, timeline 130pt
Row height: 70-80pt
Spacing between rows: thin divider line (1pt, #E0E0E0)
```

**CRITICAL: Action items must be structured, not prose paragraphs.** Each item needs:
- Clear action statement (bold, 14pt) — what to do
- Impact/rationale (regular, 12pt, grey) — why it matters
- Owner name and function (right-aligned)
- Timeline with specific date
- Never write a paragraph — decompose into these structured fields
