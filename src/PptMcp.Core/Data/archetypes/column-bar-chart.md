# 2. Column/Bar Chart

**When:** Comparing values across categories or showing trends.

**Layout:**
```
Title: top
Chart: left 60% of content area (x=50pt, y=100pt, w=550pt, h=350pt)
Key insight callout: right 35% (optional)
Source: bottom
```

**Rules:**
- Column for categories, bar for ranked lists
- Maximum 7 bars/columns
- Always label axes
- Sort bars by value (descending) unless time-series
- Never use 3D effects
