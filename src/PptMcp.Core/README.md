# PptMcp.Core

PowerPoint-specific business logic for PptMcp. Provides commands for slides, shapes, text, charts, tables, animations, transitions, design, export, VBA, and more.

This is an internal library used by [PptMcp.McpServer](https://www.nuget.org/packages/PptMcp.McpServer) and [PptMcp.CLI](https://www.nuget.org/packages/PptMcp.CLI). It is not intended for direct consumption.

## Requirements

- Windows with Microsoft PowerPoint installed
- .NET 9.0+
